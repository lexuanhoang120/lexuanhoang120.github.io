Electricity Consumption Forecasting

CADS - FPT Corporation · 2023.05 - 2023.10

Monthly electricity consumption forecasting project for 13 provinces in Vietnam using decomposition-based time-series modeling.

Demo

The demo illustrates the full forecasting pipeline with synthetic historical data (not real production data). It supports both T+1 forecasting (next month) and recursive T+12 forecasting (next 12 months).

BDINH Component Decomposition

Approach

The forecasting method decomposes each province series and predicts each component separately:

  • Trend: linear model for long-term movement.
  • Seasonality: statistical model with parameter controls (lag, order1, order2, shift).
  • Residual: ARIMA model for remaining short-term variation.
  • Final prediction: trend + seasonal + residual.

Main scripts and assets from the project README:

  • main/forecast_t1_13pr_sum.py for T+1 forecasting.
  • main/forecast_t12_13pr_sum.py for recursive T+12 forecasting.
  • main/helper.py for support functions.
  • jupyter-notebook/CPC_Forecast.ipynb for notebook walkthrough.

Reported development-time results: MAPE below 6% across all 13 provinces, and below 4% for 6 provinces (evaluation period: January 2022 to July 2023).
