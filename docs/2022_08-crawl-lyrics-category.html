Crawl Lyrics Category

VTCODE Company · 2022.08

A Selenium-based crawler that searches for a song by its title and singer, then collects the lyrics and category information from music websites.

GitHub View Approach

Demo Preview

Approach

  • Used Selenium to automate browser actions.
  • Searched songs based on title and singer name.
  • Opened the most relevant song page.
  • Extracted lyrics and category information.
  • Cleaned and displayed the final result.

Crawling Demo

The crawler opens the browser, searches for the song, and collects the needed information from the result page.

Setup

  • Install packages with pip install -r requirements.txt.
  • Download a webdriver that matches your browser version.
  • Open and run the .ipynb file with Jupyter Notebook.

Notes

  • The webdriver version should match your browser version.
  • Website XPath values may change over time.
  • If the crawler stops working, inspect the website structure again.
