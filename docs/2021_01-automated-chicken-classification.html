Automated Chicken Gender Classification System

HCMUT Control & Automation Lab · 2021.01 – 2022.06 · Undergraduate Thesis

Complete mechatronic system for automatically sorting chickens by gender. Built from scratch: self-collected the dataset on a local farm, trained a custom CNN (4 conv + 2 dense), designed the mechanical singulation mechanism in CAD, wired the ATMega 2560 control system, and ran live experiments. Multi-frame majority voting boosted accuracy from 91.5% to 95.9%. Dataset and model published on Kaggle.
