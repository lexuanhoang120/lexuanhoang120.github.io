Automated Chicken Classification System

HCMUT Control and Automation Lab · 2021.01 - 2022.06

A thesis project focused on analyzing and designing an automated chicken gender classification system. The project combines computer vision, self-collected farm image data, model training and evaluation, mechanical design for separating chickens individually from a flock, and electrical/control design for supporting the sorting mechanism.

Demo

The demo presents the full workflow of a chicken gender classification system, from collecting raw chicken images at a farm to training a classification model and connecting the prediction output with a proposed mechatronic sorting system.

Demo Flow

  1. Collect raw chicken images from a real farm environment.
  2. Build and label the dataset for chicken gender classification.
  3. Preprocess images and prepare train/evaluation data.
  4. Train an image-based classification model.
  5. Evaluate the model using experiment results and performance analysis.
  6. Use the classification output as the decision signal for the sorting system.
  7. Route the chicken through a mechanical separation model controlled by the electrical system.

Demo Highlights

  • Problem: classify chicken gender automatically from image data.
  • Data: custom dataset collected directly from a farm.
  • AI component: image-classification model for chicken gender prediction.
  • Mechanical component: mechanism design for separating chickens one by one from a flock.
  • Electrical component: electrical/control model for actuating the mechanical sorting system.
  • Application: reduce manual classification effort and support poultry-farm automation.

Approach

Project Objective

The objective of this thesis was to design an end-to-end system for chicken gender classification. Instead of focusing only on a machine learning model, the project also considered the physical automation system required to separate chickens, capture or process visual information, classify gender, and control sorting behavior.

Classification Problem

The core AI task is an image classification problem: given an input image of a chicken, the system predicts the chicken's gender. This classification result is then used as an input signal for the downstream mechanical sorting process.

Implementation Planning

  • Defined the classification problem and system requirements.
  • Collected real-world chicken images from a farm.
  • Prepared the dataset for training and evaluation.
  • Designed the model structure and experiment plan.
  • Evaluated classification performance and documented results.
  • Proposed improvements for future model and system development.
  • Designed mechanical and electrical models for the complete automation concept.

Dataset Construction

The dataset was collected manually from a farm, making the project closer to a real-world applied computer vision problem. The dataset-building process included collecting raw images, preparing usable samples, organizing data for training, and documenting the dataset for reuse.

Model Structure

The project includes a model structure for chicken gender classification using image input. The model pipeline covers data preparation, training, inference, evaluation, experiment comparison, and result analysis.

Evaluation and Experiments

  • Evaluated model predictions on prepared chicken image data.
  • Compared experimental results to understand model behavior.
  • Analyzed classification performance and error cases.
  • Documented improvement directions for better accuracy and robustness.

Mechanical Model

The mechanical part of the thesis proposes a system for separating chickens individually from a flock. This is important because the classification system needs each chicken to be processed separately before the sorting decision can be applied.

  • Designed a mechanism to guide chickens one by one through the system.
  • Considered the physical movement required before classification and sorting.
  • Connected the classification result with the sorting direction of the mechanism.

Electrical and Control Model

The electrical model supports the operation of the mechanical system. It provides the control logic required to trigger actuators or sorting components based on the predicted chicken gender.

  • Designed the electrical/control concept for the mechanical model.
  • Linked model inference output to control decisions.
  • Planned the automation flow from image classification to physical sorting.
