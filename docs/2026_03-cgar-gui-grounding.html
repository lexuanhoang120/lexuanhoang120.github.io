CGAR: Confidence-Guided Adaptive Refinement for GUI Grounding

Sejong University VLI Lab · 2026.03 · First Author · Previously submitted to CVPR 2026

Early-stage research on training-free adaptive refinement for GUI grounding. CGAR was the initial iteration that explored using confidence signals from attention maps to control when and how to refine GUI element localization. Although rejected from CVPR 2026, the core ideas evolved into DART (Density-Aware Adaptive Refinement Technique) — which achieved top-1 ScreenSpot-Pro performance and is now under review at IEEE Access.

GitHub Evolution → DART

The rejection from CVPR 2026 provided valuable reviewer feedback that led to the key revision in DART.
