Back-end Architecture (my work)

LayerTechnology
FrameworkFastAPI (Python), Uvicorn, port 9000
DatabaseMongoDB
Face detectionMTCNN (PyTorch) — rejects 0 or 2+ face images at API level
Face embeddingsVGGFace2 / ResNet50 (Keras) — async via FastAPI BackgroundTasks
TrackingCustom centroid tracker (Euclidean distance)
AuthSHA-256 hashed passwords, token-based

Key Design Decisions

  • Async embeddings: VGGFace2 runs in background — HTTP response returns immediately.
  • Single-face enrollment: Detection rejects multi-face uploads.
  • Consistent API format: All endpoints return {data, errorCode, message, errors}.
