Google Maps Business Scraper

CADS - FPT Corporation · 2024.02

A Selenium-based crawling project for collecting business phone numbers from Google Maps using keyword, ward, and district search combinations.

GitHub View Approach

Project Overview

The project reads search inputs from an Excel file, builds Google Maps search URLs, crawls business results, extracts phone numbers, and saves the data into structured CSV files.

Approach

  • Read keywords, wards, and districts from dvdc.xlsx.
  • Built Google Maps search links for each location-based query.
  • Used Selenium Chrome to open pages and scroll through result lists.
  • Extracted phone numbers using XPath selectors.
  • Saved results into separate CSV files for each search combination.

Results

  • Generated 16,623 CSV files from Google Maps crawling.
  • Collected 364,191 phone-record rows from Google Maps outputs.
  • Found 108,675 unique phone values in the main output folder.
  • Across all CSV outputs, collected 709,481 rows and 183,939 unique phone values.
