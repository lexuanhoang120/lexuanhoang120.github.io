Google Maps Business Scraper

CADS - FPT Corporation · 2024.02

Large-scale web scraping toolkit that collected 709K+ business records and 184K unique phone numbers from Google Maps, Chotot.vn, and LinkedIn. Used a geographic coverage strategy — iterating through every ward + district combination in Vietnam to maximize coverage.

How it works

  • Geographic coverage: For each keyword (e.g. "tạp hóa", "bách hóa"), the scraper iterates through all ward + district pairs from a reference file (1,152 rows) — producing 1,152 unique Google Maps searches per keyword.
  • Fast mode: Grabs visible phone numbers directly from search results via XPath.
  • Slow mode: Clicks into each business listing to extract full metadata — categories, ratings, reviews, address, place ID, phone.
  • Chotot scraper: Navigates category pages, collects listing links, visits each one with BeautifulSoup + Selenium to pull seller phone, name, address, online status.
  • Multiprocessing: Parallel browser workers for throughput.
  • Resume support: Skips already-completed ward/district combinations on restart.
