Face Recognition Attendance System

VTCODE Company · 2022.06 – 2023.03

Real-time face recognition system that automatically clocks employees in and out using an RTSP office camera. Detects faces with SSD, tracks them across frames with a custom centroid tracker, matches against stored VGGFace2/ResNet50 embeddings via cosine similarity, logs to SQLite, and greets employees by name with Vietnamese text-to-speech.

Pipeline

  • Detection: SSD-based face detector (OpenCV DNN + Caffe model) scans each frame. A separate MobileNetV2 classifier detects face masks — masked faces get a stricter matching threshold.
  • Tracking: Custom centroid tracker maintains identity across consecutive frames. Faces that disappear for 10+ frames are deregistered. Without tracking, the same person would be re-identified every frame.
  • Recognition: Detected faces → 224×224 → VGGFace2/ResNet50 → 2048-dim embedding. Compared against pre-computed employee embeddings via cosine distance. Closest match below threshold wins.
  • Attendance logic: No checkin today? → insert checkin. Checkin exists, no checkout? → insert checkout. Both exist? → update checkout timestamp.
  • Voice greeting: Vietnamese TTS via Windows SAPI. Morning: "Xin chào [name]". Afternoon: "Tạm biệt [name]".

Results

  • ~97% recognition accuracy in office deployment.
  • Real-time performance on a single machine with RTSP camera input.
  • Automatic check-in/checkout eliminated manual attendance tracking.
  • Embeddings pre-computed from 1–3 photos per employee — no per-person model training needed.
