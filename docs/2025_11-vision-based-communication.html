Vision-based Communication with Object Tracking

Sejong University VLI Lab · 2025.11

Vision-guided predictive beamforming for vehicular communication. Uses YOLOv11 for vehicle detection, ByteTrack for tracking, and linear position prediction to overcome perception-to-actuation latency. Evaluated on the DeepSense 6G dataset across 3 real-world driving scenarios, improving average data rate by +20–29% over conventional delayed-detection beamforming.

Pipeline

  1. YOLOv11 detects vehicles in each camera frame (car class)
  2. ByteTrack maintains identity across consecutive frames
  3. Linear predictor estimates the future bounding box from recent motion history
  4. 2D→3D mapping converts image coordinates to azimuth/elevation angles
  5. Angular error computed between predicted and ground-truth beam directions
  6. Antenna gain estimated from misalignment
  7. Data rate computed across SNR values (0–30 dB)

Key insight: The baseline steers the beam toward the last known position — it always lags. The proposed method forecasts where the vehicle will be when the beam forms, cutting pointing error and preserving gain.
