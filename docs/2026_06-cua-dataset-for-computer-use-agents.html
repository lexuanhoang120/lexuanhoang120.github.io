A Dataset for Computer-Use Agents

Sejong University VLI Lab · 2026.06 – Present · First Author · Target: AAAI 2027

Building and curating a dataset tailored for computer-use agent grounding and interaction tasks. This is an active research project — the dataset focuses on diverse GUI scenarios for training and evaluating agents that interact with real computer interfaces.

GitHub (coming soon) Details

Research Scope

  • Domain: Computer-use agents — AI systems that interact with GUIs to complete user tasks.
  • Focus: Grounding (locating UI elements from natural language instructions) and interaction diversity.
  • Status: In preparation. Dataset collection, annotation, and quality control in progress.
  • Target venue: AAAI 2027.
  • Role: First author — leading data collection, annotation design, and benchmark construction.

More details will be added as the project progresses toward submission.
